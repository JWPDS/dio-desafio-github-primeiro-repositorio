O Git Bash é um terminal extendido para otimizar o uso do Git.

cd -> Acessar pasta

git status -> Verificar status do Git

git add . -> Enviar pasta para controle de versão

git commit -m "....." -> Adicionar novos commits ao Git

git push origin main -> Enviar todos os commits que estão local para nuvem
